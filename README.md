# EE671 VLSI Design Projects

This repository contains my VLSI design course projects from EE671 at IIT Bombay.

## Projects Included

1. **Standard Cell Design and Characterization**
   - Transistor sizing to match inverter delay targets
   - Layout creation and physical verification
   - Parasitic extraction, timing & power analysis
   - Verilog functional views for standard cell library integration

2. **4×4 Max‑Pooling Accelerator Hardware**
   - RTL design of a hierarchical comparator tree in Verilog
   - RTL simulation and waveform verification
   - Logic synthesis, area & power reports
   - Full physical design (DRC‑clean) and post‑route timing & power

## Tools Used
NGSpice, Magic VLSI, Verilog, GTKWave, Cadence Innovus

📄 PDF project reports included in this repo for each project.
